# Data-Analysis
Customer Churn Prediction and Analysis: Built a telecom churn prediction system using data cleaning, EDA, feature engineering, and visualization. Developed Logistic Regression and CatBoost models to identify churn drivers, improve prediction accuracy, and generate actionable insights for customer retention and revenue optimization
Customer Churn Analysis and Prediction

Overview
This project looks at customer churn data and builds machine learning models to predict whether a customer will leave a service. The process includes cleaning data, exploring it, preprocessing features, visualizing results, and modeling predictions.

Objectives
- Understand the factors that lead to customer churn.
- Clean and preprocess the data.
- Visualize customer behavior and churn patterns.
- Build and evaluate machine learning models for predicting churn.

Dataset Features
- Customer Demographics
- Tenure
- Monthly Charges
- Total Charges
- Contract Information
- Payment Methods
- Internet Services
- Churn Status (Target Variable)

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- CatBoost

Project Workflow
1. Data Loading and Inspection
2. Handling Missing Values
3. Cleaning and Transforming Data
4. Exploratory Data Analysis (EDA)
5. Feature Engineering
6. Model Training
7. Model Evaluation

Visualizations

Machine Learning Models

Results

The model identifies customers at risk of churning. This lets businesses take proactive steps to retain customers and boost satisfaction.
